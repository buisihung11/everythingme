# Overview

This example documents how a re-event-the-wheel project should be organized in the EverythingMe monorepo.

## Goal

Pick an existing application and rebuild it with your own architecture and technology choices.

## Structure

- `lessons/` — running log of implementation notes and findings
- `src/` — application source (added when you start building)
- `package.json` — project dependencies
