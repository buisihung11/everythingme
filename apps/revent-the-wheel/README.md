# Re-event the Wheel

Projects where we rebuild existing applications with our own solutions.

## Layout

Each project lives under this category folder with code and lessons together:

```text
apps/revent-the-wheel/<project-name>/
├── lessons/          # required — findings, decisions, implementation notes
├── src/
├── package.json
└── project.json
```

## Getting started

1. Copy `example/` to a new folder name (e.g. `todo-clone/`).
2. Scaffold a TanStack Start app inside the new folder.
3. Document your journey in `lessons/` as you build.

See `example/lessons/` for the documentation template.
