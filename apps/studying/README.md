# Studying

Small applications to learn new technology hands-on.

## Layout

Each study project keeps documentation next to its code:

```text
apps/studying/<project-name>/
├── lessons/          # required — what you learned and how you applied it
├── src/
├── package.json
└── project.json
```

## Getting started

1. Copy `example/` to a new folder named after the technology or experiment.
2. Build the smallest app that exercises the new tech.
3. Write lessons as you go — the goal is understanding, not polish.

See `example/lessons/` for the documentation template.
