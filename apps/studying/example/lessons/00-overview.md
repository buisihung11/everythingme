# Overview

This example documents how a studying project should be organized in the EverythingMe monorepo.

## Goal

Learn a new technology by building a small, focused application.

## Structure

- `lessons/` — notes on concepts, APIs, and hands-on findings
- `src/` — experiment app source (added when you start building)
- `package.json` — project dependencies

## Next steps

1. Pick a technology to explore.
2. Create a new folder sibling to `example/`.
3. Write `lessons/00-overview.md` with learning objectives before coding.
